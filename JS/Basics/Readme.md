##Variables
- var: Function-scoped
- let: Block-Scoped; can be updated but not redeclared in the same scope.
- const:Block-Scoped;  must be initialized and cannot be reassigned.

##Data Types
- Primitive : string, number, boolean, null, undefined, symbol, bigint

##Operators
- Arithmetic: +, -, *, /, %
- Comparison: ==, ===, !=, <, >
- Logical: &&, ||, !
- Assignment: =, +=, -=, etc
